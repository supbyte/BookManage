#### 安装教程

1. 开发环境：Windows 11
2. 开发工具：IDEA 2021 (Ultimate)
3. JDK版本：17.0.10
4. 构建工具：Maven 3.6.3
5. 数据库版本：MySQL 8.0
6. 数据库连接工具：Navicat For MySQL 10.1.7
7. 服务器：Tomcat 10.1.19
8. 调试工具：Microsoft Edge

#### 使用说明

1. 运行数据库脚本导入项目所使用的数据库
2. 使用IDEA导入项目
3. 连接数据库，将数据源配置修改为自己的用户名与密码
4. 部署Tomcat服务器
5. 启动项目
